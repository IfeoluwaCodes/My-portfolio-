Learning Notes

Text

Most text on a webpage is written using HTML text tags. The <p> tag is used for paragraphs. The <h1> to <h6> tags are used for headings.

Images

Images are added using the <img> tag. It uses attributes like src and alt.

Links

Links are created using the <a> tag. The href attribute defines the destination.

Containing Elements

The <div> tag is used to group elements. CSS classes are used to style grouped content.

Navigation

Navigation is built using the <nav> tag and usually contains links to sections or pages.
